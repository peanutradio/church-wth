# 🤝 관리자 권한 공유 및 인계 가이드

이 문서는 프로젝트의 모든 권한(홈페이지, GitHub, Supabase, Vercel)을 다른 관리자에게 공유하는 방법을 설명합니다.

## 1. 🏠 홈페이지 관리자 (Website Admin)
웹사이트 내부의 관리자 페이지(`/admin`) 접근 권한을 부여합니다.

1. **[초대받을 분]** 홈페이지([https://church-wth.vercel.app](https://church-wth.vercel.app))에 접속하여 **로그인/회원가입**을 완료합니다.
2. **[현재 관리자]** 홈페이지에 관리자 계정으로 로그인합니다.
3. **[현재 관리자]** `/admin` 페이지로 이동하여 **"회원 관리"** 탭을 클릭합니다.
4. **[현재 관리자]** 목록에서 초대할 분을 찾아 **권한(Role)** 드롭다운을 `Guest` 또는 `Member`에서 **`Admin`**으로 변경합니다.
5. **[완료]** 이제 초대받은 분도 로그인 시 상단에 "관리자" 메뉴가 나타납니다.

---

## 2. 🐙 GitHub (소스 코드 저장소)
소스 코드를 수정하고 버전을 관리할 수 있는 권한을 부여합니다.

1. **[현재 관리자]** GitHub 저장소 페이지로 이동합니다.
   - URL: `https://github.com/peanutradio/church-wth`
2. 상단 탭에서 **Settings**를 클릭합니다.
3. 왼쪽 사이드바에서 **Collaborators**를 클릭합니다.
4. **Add people** 버튼을 클릭합니다.
5. 초대할 분의 **GitHub Username** 또는 **이메일**을 입력하고 `Add ... to this repository`를 클릭합니다.
6. **[초대받을 분]** 이메일로 발송된 초대장을 수락합니다.

---

## 3. ⚡ Supabase (데이터베이스 & 백엔드)
데이터베이스, 인증(로그인), 스토리지 등을 관리할 수 있는 권한을 부여합니다.

1. **[현재 관리자]** [Supabase Dashboard](https://supabase.com/dashboard)에 접속합니다.
2. 왼쪽 하단의 **Organization Settings** (건물 아이콘)을 클릭합니다.
   - *참고: 프로젝트 설정이 아니라 조직(Organization) 설정으로 가야 합니다.*
3. 왼쪽 메뉴에서 **Members**를 클릭합니다.
4. 우측 상단의 **Invite Member** 버튼을 클릭합니다.
5. 초대할 분의 **이메일 주소**를 입력하고 초대장을 발송합니다.
6. **[초대받을 분]** 이메일 초대장을 수락하고 Supabase에 가입/로그인합니다.

---

## 4. ▲ Vercel (배포 & 호스팅)
웹사이트 배포, 도메인 연결 등을 관리합니다.

> **⚠️ 주의: 무료 요금제(Hobby) 제한**
> Vercel의 개인 무료 요금제(Hobby)를 사용 중인 경우, **팀원 추가 기능이 제한**될 수 있습니다. (팀원을 추가하려면 유료 Pro 플랜으로 업그레이드해야 할 수 있음)

### 추천 방법 (GitHub 연동 활용)
Vercel 권한을 직접 주지 않아도, **GitHub 권한(2번)**만 있으면 배포 관리가 가능합니다.
- 코드를 수정해서 GitHub에 `push`하면 Vercel이 자동으로 감지해서 배포합니다.
- 따라서 개발/유지보수 목적이라면 GitHub 권한만으로 충분합니다.

### 꼭 Vercel 권한이 필요한 경우
만약 도메인 변경이나 환경 변수 수정 등 Vercel 설정 자체를 만져야 한다면:
1. **[초대받을 분]** 본인의 Vercel 계정을 만듭니다.
2. **[초대받을 분]** Vercel 대시보드에서 `Add New...` -> `Project`를 클릭합니다.
3. **[초대받을 분]** `Import Git Repository`에서 공유받은 GitHub 저장소(`church-wth`)를 선택하여 **Import** 합니다.
4. **[초대받을 분]** 환경 변수(.env)를 입력하고 **Deploy** 합니다.
   - *주의: 이렇게 하면 별도의 배포 주소가 생성됩니다.*
